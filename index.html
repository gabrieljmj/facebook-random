Exemplos com URLs

  • https://www.facebook.com/photo.php?fbid=1111111111111&set=a.22222222.3333333.444444444444&type=1
    O parâmetro fbid é o ID. No caso, 1111111111111.
  • https://www.facebook.com/page-name/photos/a.123456.654321.7890123/123456654321/?type=1
    O ID é 123456654321.
  • https://www.facebook.com/photo.php?v=10110001&set=a.22222222.3333333.444444444444&type=1
    O parâmetro v é o ID para posts de vídeos. No caso 10110001.
  • https://www.facebook.com/page-name/posts/123456789
    O ID em posts de status (sem arquivos de mídia - fotos, vídeos). 123456789.
